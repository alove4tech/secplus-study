# Security+ Command Center

A responsive static study application for CompTIA Security+ SY0-701 prep.

## What is included

- Dashboard with readiness score, daily plan, domain progress, quiz shortcut, flashcard streak, and featured lab.
- Practice quiz flow with a 300-question SY0-701 question bank, answer feedback, score tracking, random question mode, and explanations.
- Performance-Based Questions simulator with configurable PBQ-style tasks and grading.
- Flashcard review flow.
- Lab Simulator with evidence, containment checklist, hints, scoring, and debrief.
- Study Plan, Notes, Mock Exams, Settings, and Progress views.
- Dark mode toggle.
- Responsive desktop and mobile layout.

## Run locally

Open `index.html` in a browser.

Because this is a static app, it can be hosted on GitHub Pages, Netlify, Vercel, Cloudflare Pages, or any basic web host.
