# Security Policy

## Reporting a Vulnerability

If you find a security issue in this project, please report it responsibly:

- Open a [GitHub issue](https://github.com/alove4tech/secplus-study/issues) and tag it as a security concern
- Or contact the maintainer directly through GitHub

Please do not publicly disclose vulnerabilities before a fix is available.

## Scope

This policy covers the application code and Docker configuration in this repository. It does not cover third-party dependencies — report those to the respective upstream projects.

## Data

All user data is stored in localStorage. No data is transmitted to any server. There are no accounts, databases, or telemetry.
